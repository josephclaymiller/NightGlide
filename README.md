# Night Glide

Night Glide is a platformer game developed collaboratively by undergraduate and graduate students from a variety of majors through the Game Development Club. The game challenges players to survive as a flying squirrel — navigating predators, foraging for food, and searching for love. This project shows that with curiosity, collaboration, and a willingness to learn, students from any background can take on the rewarding challenge of building a game from scratch.
